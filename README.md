# Dark-Wanderer-Class
A mod for D2RLAN that replaces the Sorceress Class with the Dark Wanderer who can transform into Diablo when the Diablo's Soulstone is Equipped. "Play as Diablo" original mod is by Emuhel on Nexus mods. Divinakra ported that mod to D2RLAN and added the Dark Wanderer and Diablo's Soulstone features, and improved Diablo's Skills.
