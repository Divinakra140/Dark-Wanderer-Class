My original intent was simply just to port Emuhel's Play 
as Diablo Mod to D2RLAN, so we could do some fun online
Diablo vs Diablo PVP games, but then I ended up wanting 
to add some QOL features and improvements and then ended 
up adding much more than that, resulting in a new base class
that lines up canonically with the transformation into 
Diablo; The Dark Wanderer, aka Prince Aidan.

Note: these instructions are for manual installation:
(otherwise select it from the D2RLAN Dropdown menu,
where it says "Download New Mod")

Mod Manual Installation Instructions:
1. Download the .zip file called dark_wanderer_d2rlan.zip 
2. Unzip the file. it should result in a "mods" folder.
3. Place that "mods" folder in your D2RLAN Directory 
it should sit right next to the D2R.exe 
4. Set your launch parameters on the D2R.exe to this: 
-mods Dark -txt
that will tell the game to use the mod during launch.
5. Double Click D2R.exe and you should see the Dark 
Wanderer as a class you can select. 
6. Adding the Launch Parameters -mods Dark -txt will
create a path to a new save folder, so your old saves 
will not appear. However if you want to use an old save
you can drag in your old save files (.D2s) into this folder:
C:\Users\your username\Saved Games\Diablo II Resurrected\mods\Dark
Note* Not all save files will work, so test them out and see
8. In game, to obtain the Diablo's Soulstone, you can either:
Buy it from Akara in Act 1 or transmute antidote + key in cube.

This mod started out as a simple D2RLAN port of Emuhel's "Play 
as Diablo" mod made for D2RMM found on Nexus. He agreed to me
making a D2RLAN port of the mod. The task of converting 
his mod to a D2RLAN compatible mod was difficult in and of itself,
due to the different structure of the D2RLAN files. The tool 
D2Compare created by Bonsey and updated by DazUki available 
on the D2R-Modding Discord made the process a lot faster! 

<img width="847" height="657" alt="Screenshot 2026-07-18 at 3 16 21 AM" src="https://github.com/user-attachments/assets/6ff5b623-5e32-4244-8ee9-16cddd0406e8" />


Once completed, I noticed some things were not quite to my 
liking; the small charm that turned the player into Diablo
was just a plain charm, nothing significant about it and 
was functional but not memorable or fun. I also noticed the
Diablo skills were not 1:1 with Boss Diablo and one was a
player skill that was modified slightly to appear as a boss 
Diablo Skill. I also did not like that I could not access 
the Diablo state until Act 2, due to the only way to obtain 
the small charm being a cube recipe. So I updgraded the mod 
pretty signififcantly in all above ways as well as added 
brand new features such as a base class called the Dark 
Wanderer with his own animations, particles and custom 
base skill of summoning Vile Children that has never been 
seen anywhere in the game other than the canon encounter 
with the Dark Wanderer in Act III.

<img width="845" height="1829" alt="Screenshot 2026-07-18 at 2 58 13 AM" src="https://github.com/user-attachments/assets/cde22390-4e59-4ce9-8d88-5c6c84b337fe" />


My goal is to make you feel like the Dark Wanderer and 
experience a true-to-lore transition into Diablo. I am
not interested in making this mod balanced or have a slow
progression. Primevial forces are supposed to overpowered 
and I did my best to maintain the primeval potency of the 
Dark Wanderer and Diablo, instead of turning them into 
mortals. I want you to feel that raw power when playing.
